# ReliefWeb Mobile v2

New version of the ReliefWeb mobile site built using [React](https://facebook.github.io/react/) and [Next.js](https://github.com/zeit/next.js).

## Running in dev mode

`npm run dev`

## Running in production mode

```
npm run build
npm run start
```

## Linting

https://standardjs.com/

`npm run lint`
